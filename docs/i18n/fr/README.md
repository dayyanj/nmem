**Mémoire cognitive pour les agents IA** : hiérarchique, auto-révisable et indépendante du framework.

nmem donne à vos agents un cerveau qui apprend. Pas seulement le stockage et la récupération, mais une cognition active — promotion automatique, révision des croyances, rétrospectives nocturnes, apprentissage social entre agents, et injection de prompt suivie des tokens.

> **Nous recherchons activement des contributeurs.** nmem est intentionnellement construit comme un projet communautaire. Le code, les documents, les idées, les rapports de bogues, les benchmarks indépendants et les tests adversaires sont tous les bienvenus. Voir [Contribuer](#contribuer) ci-dessous.
