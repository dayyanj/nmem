**Memória cognitiva para agentes de IA**: hierárquica, auto-refinada e independente de framework.

nmem dá aos seus agentes um cérebro que aprende. Não apenas armazenamento e recuperação, mas cognição ativa — promoção automática, revisão de crenças, retrospectivas noturnas, aprendizado social entre agentes e injeção de prompt rastreada por token.

> **Estamos ativamente buscando contribuidores.** O nmem foi intencionalmente construído como um projeto orientado à comunidade. Código, documentação, ideias, relatos de bugs, benchmarks independentes e testes adversariais são todos bem-vindos. Veja [Contribuindo](#contribuindo) abaixo.
