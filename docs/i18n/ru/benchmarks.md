## Бенчмарк Spwig: Быстрые цифры

**Настройка:** 17-репозиторий платформа электронной коммерции, 45 тестовых задач, 5 вариантов, 225 двойных оценок судей.

| Вариант | Что это | Оценка судьи | Стоимость |
|---------|-----------|-------------|------|
| **v8_mcp** | Агент ищет nmem через инструменты MCP | 4.27/5 | **$4.35** |
| new_developer | Нет памяти, исследует с нуля | 4.36/5 | $8.18 |
| control | Автоматическая память Claude Code (82% охват фактов) | 3.98/5 | $7.06 |
| v8_injected | Память предварительно вставлена в приглашение | 4.02/5 | $15.27 |
| v8_briefing | API краткого содержания с сигналами распознавания | 3.96/5 | $18.36 |

**Ключевое наблюдение:** Поиск MCP является самым дешевым И САМЫМ ТОЧНЫМ, потому что агент решает, что искать, основываясь на каждом вопросе. Предварительная вставка предполагает, что может быть полезно, прежде чем увидеть вопрос.

## Бенчмарк здравоохранения: Быстрые цифры

**Настройка:** 180-дневная симуляция, 4 агента здравоохранения, 200 синтетических пациентов, 1705 встреч, 40 тестовых вопросов, Qwen3-14B на RTX 4090 ($0 стоимость вывода).

| Категория | nmem | Базовая линия | Дельта |
|----------|------|----------|-------|
| **Пересмотр убеждений** | **4.75/5** | 3.21/5 | **+48%** |
| Прямое воспоминание | 4.09/5 | 3.73/5 | +10% |
| Обнаружение паттернов | 3.48/5 | 3.33/5 | +4% |
| В целом | 3.84/5 | 3.60/5 | +7% |

**Ключевое наблюдение:** Пересмотр убеждений является сильнейшим отличием nmem. Когда руководства меняются, nmem обнаруживает противоречие, разрешает его при консолидации и извлекает обновленную политику. Базовая LLM не имеет механизма для различения текущих и устаревших знаний. После дня 120 (все руководства изменены), nmem получает идеальные 5.00/5 по каждому вопросу пересмотра убеждений.

## Методология

### Бенчмарк Spwig
- **Агент:** Claude Sonnet 4.6, без головы (`claude -p`), одно обращение на задачу
- **Судьи:** Qwen3-14B (GPU) + Qwen3-30B-A3B MoE (CPU), независимая оценка, шкала 1-5
- **Изоляция:** Отдельный PostgreSQL на порту 5435, отдельно от разработки
- **Корпус:** 6076 записей — разговоры, дистиллированные LLM + семантически разбитая автоматическая память + документы + git
- **Контроли:** Чистый HOME для new_developer, нет сохранения сессии, та же база CLAUDE.md

### Бенчмарк здравоохранения
- **Модель:** Qwen3-14B-AWQ на vLLM (один RTX 4090, $0 вывод)
- **Судья:** Qwen3-14B, шкала 1-5 по критериям
- **Данные:** Синтетические пациенты Synthea (Apache 2.0) + ручные клинические сценарии
- **Симуляция:** 180 дней сжаты в 4,6 часа через временную симуляцию
- **Оценка:** 40 вопросов в 7 интервалах (дни 1, 30, 60, 90, 120, 150, 180)
