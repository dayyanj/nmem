**Когнитивная память для агентов ИИ**: иерархическая, самонастраивающаяся и независимая от фреймворка.

nmem дает вашим агентам мозг, который учится. Не только хранение и извлечение, но и активное когнитивное поведение — автоматическое продвижение, пересмотр убеждений, ежедневные ретроспективы, социальное обучение между агентами и инъекция признаков токенов.

> **Мы активно ищем волонтеров.** nmem специально разработан как проект, управляемый сообществом. Приветствуются код, документация, идеи, отчеты об ошибках, независимые тесты и агрессивное тестирование. См. [Contributing](#contributing) ниже.
