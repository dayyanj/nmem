**AIエージェント用の認知メモリ**: 階層的、自己修復性、フレームワーク非依存。

nmemはエージェントに学習する脳を提供します。単なる保存と検索だけでなく、アクティブな認知——自動的な昇格、信念の修正、夜間の振り返り、エージェント間の社会的学習、トークン追跡付きのプロンプト注入。

> **私たちは積極的に貢献者を探しています。** nmemは意図的にコミュニティドリブンのプロジェクトとして構築されています。コード、ドキュメント、アイデア、バグレポート、独立したベンチマーク、対抗テストはすべて歓迎されます。下記の[Contributing](#contributing)を参照してください。
