**AIエージェント用の認知メモリ**: 階層的、自己改善型、フレームワーク非依存型。

nmemは、エージェントに学習する脳を提供します。単なる保存と検索ではなく、アクティブな認知機能を備えており、自動的な昇格、信念の修正、夜間の振り返り、エージェント間の社会的学習、トークン追跡付きのプロンプトインジェクションが可能です。

> **私たちは積極的に貢献者を求めています。** nmemは意図的にコミュニティドリブンなプロジェクトとして構築されています。コード、ドキュメント、アイデア、バグレポート、独立したベンチマーク、対抗テストなど、すべて歓迎されます。下記の[Contributing](#contributing)をご覧ください。
