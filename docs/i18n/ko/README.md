**AI 에이전트용 인지 메모리**: 계층적, 자가 개선, 프레임워크 무관.

nmem은 에이전트에게 학습하는 뇌를 제공합니다. 단순한 저장 및 검색을 넘어, 자동 승격, 신념 수정, 밤샘 회고, 에이전트 간 사회적 학습, 토큰 추적된 프롬프트 주입을 포함한 능동적 인지 기능을 제공합니다.

> **우리는 현재 기여자를 적극적으로 찾고 있습니다.** nmem은 커뮤니티 중심으로 의도적으로 설계되었습니다. 코드, 문서, 아이디어, 버그 리포트, 독립적인 벤치마크, 적대적 테스트 등 모든 것이 환영됩니다. 아래 [기여하기](#contributing)를 참조하세요.
