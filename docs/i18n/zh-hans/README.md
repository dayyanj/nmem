**AI代理的认知记忆**：分层、自我优化、与框架无关。

nmem 为您的代理提供了一个学习的“大脑”。不仅仅是存储和检索，而是主动认知——自动晋升、信念修订、夜间回顾、跨代理的社会学习以及跟踪令牌的提示注入。

> **我们正在积极寻找贡献者。** nmem 有意作为社区驱动的项目构建。欢迎提交代码、文档、想法、错误报告、独立基准测试和对抗性测试。请参见下方的 [贡献指南](#contributing)。
